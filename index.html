Quad

QUAD provides a comprehensive overview of the memory access behavior of an application.

View project onGitHub

Basic Description

QUAD (Quantitative Usage Analysis of Data) provides some useful dynamic information regarding data usage between any pair of co-operating functions/basic blocks in an application. Various data transfer related statistics are calculated in the sense of Producer-Consumer binding. QUAD reports these statics in a DOT file for visualization as well as in XML format to provide input to some other tool.
