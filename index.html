QUAD

QUAD provides a comprehensive overview of the memory access behavior of an application.

View project onGitHub

Basic Description

QUAD (Quantitative Usage Analysis of Data) provides dynamic information regarding data usage between any pair of co-operating functions in an application. Various data transfer related statistics are extracted from the actual execution of the application in the sense of Producer-Consumer bindings. Though primarily designed to detect the actual data dependencies at the function level, QUAD is also able to provide these information at finer granularity level of user-defined blocks inside functions. The extracted information are reported in the format of a DOT file for graphical visualization as well as in XML format, which can be easily fed into other tools upon requirement.
